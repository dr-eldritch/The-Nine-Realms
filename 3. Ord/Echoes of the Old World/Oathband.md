Oathbands are the minds of the original seven KNefilimb, built to fight the archangels in the Fourth Holy War. 

1. Kromos - can change anythings' colour / hide
2. Klastos - can reshape anything / morph
3. Khrysos - can convert anything to gold / form
4. Kryos - can stop anything from moving / balance
5. Klavos - can open anything / learn - in the Klavos Crypt in Invicta
6. Kosmos - can join anything together, metal / bind
7. Oriel (Kaustos) - can set anything on alight / ruin - is on Dogu
